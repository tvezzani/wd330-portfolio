CSS Transforms and Transitions

Transforms allow an otherwise static page to have some real time changing elements.

Transforms

The transform property allows you to manipulate elements and text on a page through rotation, scale, color, and other properties.

Translation

Translation allows you move elements by position and can help you move to different relative or absolute values. Some older browsers do not support full translation and require plugins.

Scaling

Scaling allows you to change the x and y components of scale. You can stretch an image if you don’t change them both relative to each other. It is implemented through transform: scale(1.5, 0.25);

Rotation

Rotation is similar to scale, however it accepts degrees like so: rotate(10deg). Rotation can be a useful function for breaking the norm the document.

Skew

Skew is implemented through skew(x,y), where x and y are degrees of skewing. It can stretch and squash an element.

Changing the Origin of the Transform

The transform-origin property can be used to change the origin, which can help by changing the pivot point of relative transformations. Using this, you can rotate or skew an element in just about any way imaginable.

Transitions

Transitions can be used with transformations and add a lot more to the web page. For instance, a transformation can be used to animate transform features and make the page really stand out.

The transition-duration Property

The transition duration allows you to set a time during which the transition will take place. For instance, transition-duration: 1s; will have a transition take place in 1 second.

The transition-timing-function Property

With this property you have even more control over the transition, with timing functions like ease, linear, ease-in, ease-out, etc.

The transition-delay Property

This one is self-explanatory. The transition-delay property allows you to set a time before the transition begins. Usually you want a small time duration here. If a negative value is entered, then it will subtract from the transition time. For example, a 2 second transition-duration will be a 1.5 second duration if transition-delay is set to -0.5s

The transition Shorthand Property

Transition properties can be entered in order after a transition statement like so:

Transition: transform 0.5s ease-in 10ms;

transition-property, duration, timing-function, and delay are represented here, respectively.

Multiple Transitions

You can add transitions back-to-back to cause multiple transitions to occur at the same time, allowing you to layer several different animations and transforms.

Animations

Animations take it a step even further and introduce keyframes, giving you full control over an animation over elements in the page.

Keyframes

Keyframes use key terms like from and to which specify actions at different intervals over the animation.

Animation Properties

The following are animation properties: animation-name and animation-duration, which determine the name of the animation and the time it takes to complete.

animation-timing-function

Animation timing can be linear, ease, ease-in, ease-out, and ease-in-out.

animation-iteration-count

Animation iteration will default to 1, and defines the amount of times the animation will play. It can also be set to infinite, to have a forever repeating animation.

animation-direction

You can reverse the direction of the animation, making it go backwards.

animation-delay

This sets the delay to wait before playing the animation.

Canvas, SVG, Drag, and Drop

 

Canvas

This API allows you to draw in HTML5. You can manipulate pixels in images and videos.

A Bit of Canvas History

Canvas was originally developed by Apple. Later it was adopted by Mozilla and then standardized by W3C and finally HTML5.

Creating a canvas Element

You use the <canvas></canvas> tags to create a canvas element.

Drawing on the Canvas

You have to reference the canvas element with JavaScript.

Getting the Context

You must set context attributes to draw, including setting it to 2D mode.

Filling Our Brush with Color

The attributes strokeStyle and fillStyle can be set to your preference.

Drawing a Rectangle to the Canvas

A rectangle can be made with context.fillRect.

Variations on fillStyle

Fillstyle can be changed to pattern or gradient.

Drawing Other Shapes by Creating Paths

Circles can also be drawn using paths.

Saving Canvas Drawings

A drawing can be saved to a data Url.

Drawing an Image to the Canvas

An image can be drawn onto a canvas.

Manipulating Images

Images can be stretched and squashed with getImageData.

Security Errors with getImageData

Accessing the file:// protocol can cause errors.

Converting an Image from Color to Black and White

An image can be converted through manipulating imageData and setting the red, green and blue values differently to grayscale.

Manipulating Video with Canvas

You can edit a video frame by frame in real time, for example, turning it black and white.

Displaying Text on the Canvas

Text can be easily displayed on canvas through setting a font and filling text.

Accessibility Concerns

There is no DOM node and no text-based format, so disabled persons would not be able to view the canvas.

SVG

SVG stands for Scalable Vector Graphics, and is very useful for high resolution drawing.

Drawing in SVG

With the <svg> tag, you can generate shapes with ease.

SVG Filters

Filters allow you to change the color of elements and customize their look.

Using the Raphael Library

You can have access to a container to draw custom shapes using this library.

Canvas versus SVG

Because canvas is pixel based, it can be more performant if you are demanding lots of graphics. But SVG wins in scalability and quality if you don’t need too many elements on your page.

Drag and Drop

The drag and drop API allows you to drag elements across the page and drop them in predetermined areas, or freely. It can be very useful for several applications and help connect the user with the site.

Making Elements Draggable

The attribute draggable=”true” is not a Boolean, so it must be exactly “true”. It can be set on virtually any object to make it draggable.

The DataTransfer Object

The data transfer object allows use to give an id to a draggable element and help make some use out of it.

Accepting Dropped Elements

To accept a draggable element, an event listener must be enabled that listens for the dropped element and reacts to it.
