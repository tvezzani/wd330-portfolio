The Window Object

The window object is represented by the browser window containing the page.

The Browser Object Model

Known as BOM for short, it contains methods returning information about the browser and the computer screen. It can also function using the window object. For example, in the window, you can set a variable such as x = 5. This can be retrieved by calling window.x.

Alert, confirm, and prompt can be called through the window object. Alert will popup a message. Confirm will show a cancel/okay dialogue. The prompt dialogue allows the user to enter information into a popup.

Finding out browser information

You can find browser information. One example is typing window.navigator.userAgent which returns the kind of browser being used. Using href returns the direct link. Protocol returns the protocol. Host returns the site name. hostname returns the domain. Port returns the port. Pathname returns a path. Search allows you to search, using a query. Hash returns the fragment identifier if there is any. Origin returns the domains of the current page’s origin. You can redirect with window.location.assign(). toString returns a string with the whole url.

Browser history

History length shows how many pages have been visited before. Go is a function that takes you to the page based on the integer, 0 reloading the current page, 1 going forward and -1 going backward. You can also use forward() and back() to do the same thing.

Controlling windows

Window open() can open a url, with specific size and setting parameters. The close() method can close a popup. Window resizeTo() can manipulate the size of a window. AvailHeight and availWidth are helpful to read width and height. The colorDepth method returns the color depth.

The document object allows you to write to the HTML document.

Cookies

Cookies are saved locally to help sites remember settings and user preferences effortlessly. The are very small in size, limited to 4KB of data, and only 20 cookies per domain. You can create a cookie by using the document object, by using document.cookie = “some text”; Editing cookie values is as simple as rewriting the cookie. Reading them is done through entering document.cookie, and you can write code to break down the string. Cookies are automatically deleted upon session close, but they can be set to persist through the setTime(new Date) method.

Timing functions

There are functions that can count down, like the setTimeout() function. This will call when the time is up. You can clear the alert before it is called through clearTimeout() The setInterval() method will be called repeatedly. Similar to the clear timeout function, clearInterval() clears an interval timer.

HTML5 APIs

HTML5 is the latest version of Hypertext Markup Language currently available. With APIs, you can extend its use to virtually infinite applications.

The development of HTML5 and the JavaScript APIs

While there is future development planed for HTML5, it is supposed to have long term support. HTML 5.2 is under development and can be followed until it is finished.

The data- attribute

This attribute allows you to embed data in a web page. You can do this through data-name = ‘bob’. This exposed attributes to be accessed through the dataset attribute.
